xcompose-stem

Because copying ∞ from Google or whatever is ridiculous
Symbol Reference GitHub Repository

The Problem

Linux has excellent Unicode support, but typing math symbols is a mess. Your options are:

  • Copy/paste from Google (tedious, breaks flow)
  • Use Character Map apps (slow, can't remember where anything is)
  • Install LATEX and render everything (overkill for a quick email)
  • Memorize Unicode codepoints (lol no)

The Compose key is the elegant solution, but existing XCompose configs have issues:

  • Huge symbol sets (47,000+ sequences) — impossible to memorize
  • No conflict validation — sequences that shadow each other
  • Not Omarchy-aware — collide with built-in emoji shortcuts
  • Kitchen-sink approach — everything for everyone, usable by no one

The Solution

xcompose-stem is a curated, validated XCompose configuration focused on STEM symbols. Hundreds of sequences instead of thousands. Type symbols directly in any application without needing a LATEX renderer.

⎄ h 8 becomes ∞
⎄ h i t becomes ∫

Key differentiators:

  • Systematic prefixes (h for math, g for Greek) make sequences learnable
  • Dual access patterns (visual shortcuts + mnemonics) fit your workflow
  • Automated validation ensures zero conflicts
  • Omarchy-friendly by design — deliberately avoids emoji shortcut collisions
⚠️ Work in Progress
This project is actively developed. Mappings may change, be added, or be removed until version 1.0 is released.

What's a Compose Key?

The Compose key (represented by the ⎄ symbol) lets you type Unicode symbols by pressing key combinations. Think of it like a shift key for symbols:

⎄ g a becomes α
Shift 1 becomes !

Same concept, different symbols.

Most Linux desktop environments support it natively, though you'll have to enable it (see Quick Start below).

Using Omarchy Linux? You're all set! Caps Lock is already your Compose key. Just install xcompose-stem and start typing. Press Super K to see all Omarchy hotkeys.

Quick Start

1
Clone the repository
git clone https://github.com/phil-bowens/xcompose-stem.git ~/.xcompose-stem
2
Run the installer
cd ~/.xcompose-stem && ./install.sh
3
Start typing symbols
⎄ g a becomes α
⎄ h 8 becomes ∞
⎄ - > becomes →

See the installation guide for detailed instructions and platform-specific setup.
